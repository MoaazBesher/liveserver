# Universal Live Server

A Windows desktop application that automatically detects your project type and launches the appropriate development server with a single click.

---

## Download

**[Download UniversalLiveServer-1.0.0-Setup.exe](https://github.com/MoaazBesher/liveserver/releases/download/v1.0.0/UniversalLiveServer-1.0.0-Setup.exe)**

---

## Features

- Auto-detects 25+ project types and runs the correct server command automatically
- One-click start and stop — no terminal required
- Opens your browser automatically when the server is ready
- npm script selector for Node.js projects (dev, start, build, etc.)
- Dependency installer (npm install, pip install, composer install, etc.)
- Port conflict detection with option to kill the blocking process
- Drag and drop folder support
- Network address display for sharing on local network
- Dark theme UI

---

## Supported Projects

| Language / Runtime | Detected Frameworks |
|---|---|
| Node.js | React (Vite, CRA), Vue (Vite, CLI), Next.js, Nuxt.js, Angular, SvelteKit, Astro, Gatsby, Express |
| PHP | Laravel, Generic PHP, Composer |
| Python | Django, Flask, FastAPI, Generic Python HTTP server |
| Ruby | Ruby on Rails, Jekyll |
| Go | Go modules |
| C# / .NET | .NET Core / .NET 5+ |
| Docker | Docker Compose, Dockerfile |
| Static | HTML / CSS / JS via live-server |

---

## Getting Started

1. Download and run the installer above
2. A shortcut will be created on your Desktop
3. Open the app, browse to your project folder or drag and drop it onto the window
4. Click **Start Server**

---

## Requirements

- Windows 10 or 11
- .NET Framework 4.8 (pre-installed on most Windows systems)
- The relevant runtime for your project (Node.js, PHP, Python, etc.)
